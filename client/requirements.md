## Packages
three | 3D library for technical visualizations
@react-three/fiber | React renderer for Three.js
@react-three/drei | Helpers for React Three Fiber
framer-motion | Smooth, physics-based animations for UI elements
recharts | Data visualization for skills and metrics
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes efficiently

## Notes
- Tailwind config should extend fontFamily to include 'Space Grotesk', 'Inter', and 'JetBrains Mono'.
- The 3D component is a visual anchor for the Hero section, representing a neural network topology.
- Contact form submits to /api/contact.
- Projects are fetched from /api/projects.
- Skills are fetched from /api/skills.
